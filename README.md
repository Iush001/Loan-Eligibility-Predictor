# Loan Eligibility Prediction using Machine Learning

## Project Overview
This project predicts whether a loan application will be approved or rejected using machine learning techniques. The model analyzes applicant details such as income, education, credit history, and loan amount to determine loan eligibility.

The goal of this project is to build a classification model that helps financial institutions automate loan approval decisions.

---
## Objectives
- Perform data preprocessing and cleaning
- Analyze applicant financial data
- Train machine learning models
- Predict loan approval status
- Evaluate model accuracy

---

## Dataset
The dataset used for this project is taken from Kaggle's Loan Prediction Dataset.

### Features in Dataset
- Gender
- Married
- Dependents
- Education
- Self Employed
- Applicant Income
- Coapplicant Income
- Loan Amount
- Loan Amount Term
- Credit History
- Property Area
- Loan Status (Target Variable)

---

##  Machine Learning Models Used
The following models can be used for prediction:

- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)
